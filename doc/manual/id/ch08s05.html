8.5. Mengkompilasi Kernel Baru

Mengapa seseorang ingin mengkompilasi kernel baru? Ini kemungkinan besar tidak perlu karena kernel baku yang dikirimkan dengan Debian menangani hampir semua konfigurasi.

Namun, jika Anda ingin mengkompilasi kernel Anda sendiri, ini tentu saja mungkin dan kami merekomendasikan penggunaan target “make deb-pkg”. Untuk informasi lebih lanjut, baca Debian Linux Kernel Handbook.
